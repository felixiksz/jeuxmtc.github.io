CORRECTIONS ADMINISTRATEUR — CONNECTIONS MTC

ACTIVER LE MODE ADMINISTRATEUR
1. Ouvrir une fois le jeu avec ?admin=1 à la fin de l’adresse.
   Exemple : https://ton-site.github.io/?admin=1
2. Le mode reste actif dans ce navigateur. Pour le désactiver, ouvrir la page avec ?admin=0.

CORRIGER UNE FICHE
1. Ouvrir la fiche ACU ou PHARMA concernée.
2. Cliquer sur « ✎ Corriger la fiche ».
3. Corriger directement le ou les champs.
4. Il n’y a pas de bouton Enregistrer : chaque champ est sauvegardé automatiquement
   quand on le quitte, ou quand on ferme la zone de correction.

Les visiteurs qui n’ont pas activé le mode administrateur ne voient ni l’éditeur de fiche
ni le bouton 🗃️. Le champ Associations existe pour tous les points, même lorsqu’il est vide.

EXPORTER / IMPORTER LES CORRECTIONS
Le bouton 🗃️, visible seulement en mode administrateur, permet d’exporter toute la base
avec les corrections locales ou de réimporter un fichier JSON corrigé. Les identifiants
des points et substances ne doivent pas être renommés.

IMPORTANT
Ce mode administrateur est local au navigateur : il masque les outils aux visiteurs ordinaires,
mais ce n’est pas une authentification serveur. Pour publier définitivement les corrections,
il faut exporter le JSON corrigé puis l’intégrer à la base publiée.

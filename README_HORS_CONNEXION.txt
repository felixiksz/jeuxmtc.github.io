MODE HORS CONNEXION — INSTRUCTIONS SIMPLES

À faire une seule fois, avec internet, depuis le lien jeu :
https://felixiksz.github.io/jeuxmtc.github.io/

1. Ouvre le jeu depuis son lien web.
2. Choisis ce que tu veux garder sur cet appareil.
3. Ensuite, rouvre le même lien : tu peux jouer sans internet et sans distractions.

*psst : ajoute la page sur ton écran d’accueil pour des révisions quotidiennes.

NOTES POUR LA PUBLICATION GITHUB

- Mets tous les fichiers du ZIP au root du dépôt GitHub, au même niveau que index.html.
- Le mode hors connexion se prépare depuis le lien web publié, pas depuis un fichier index.html ouvert dans “Fichiers” ou “Téléchargements”.
- Le bouton ✈︎ prépare le jeu sur l’appareil.
- “Jeu seul” garde les fichiers essentiels.
- “Jeu + audios” garde aussi les prononciations, mais c’est plus lourd.
- Après une mise à jour du jeu, rouvre le lien avec internet et clique de nouveau sur ✈︎.
